# ExamGuard Landing Page

صفحة هبوط عربية RTL لشرح وبيع برنامج امتحانات أون لاين.

## الملفات
- index.html
- styles.css
- script.js

## طريقة التشغيل
افتح ملف `index.html` مباشرة في المتصفح.

## التعديل
- غيّر اسم المنتج من داخل `index.html`.
- عدّل الأسعار من قسم `pricing`.
- يمكن ربط نموذج الطلب بواتساب أو بريد إلكتروني أو API لاحقاً من ملف `script.js`.
